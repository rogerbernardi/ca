Servidor NodeJs - API POC para integrações CA PAM - CA USD.
Criado por Roger.bernardi@mphg.com.bernardi
07/11/2018

Para iniciar:

Instalar o nodejs.
Digitar: node nodeServer.js

Instalar o curl
Digitar: curl localhost:9090
Deve receber a SID do CA USD.
Valores configurados nas variaveis do nodeserver.js.
